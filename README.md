## node mpd client

**This repository is WIP.**

Connect to a [Music Player Daemon](https://musicpd.org) ([GIT](https://github.com/MusicPlayerDaemon/MPD)) server, send commands, emit events.

This is a rewrite of [mpd2 module](https://github.com/cotko/mpd.js) to support Stream API based methods and fully written in TypeScript.
